ajax请求的方式

Posted on 2019-10-27

Promise

异步调用
异步效果：
1.定时器
2.ajax
3.事件函数

多次异步调用
    1.多次异步调用的结果顺序不确定
    2.如果异步的异步存在依赖关系,只能函数嵌套来确定顺序，
    容易形成回调地狱

Promise 是异步编程的一种解决方案  
它是一个对象 可以获取异步操作的消息

fetch

更加简单的数据获取方式,功能更强大 （基于promise）

axios

axios 是一个第三方js库

支持 浏览器 和 node
基于promise

基本用法：

axios.get('http://localhost:3000/adata')
// 这里 .data 是固定的用法（用于获取数据）
.then(ret => console.log(ret.data))

常用API

- get
    1.通过url直接传递数据
    前端：
    
        app.get('/axios', (req, res) => {
          res.send('axios get 传递参数' + req.query.id)
        })
    
      axios('http://localhost:3000/axios',{
          params:{  id=wxz }
          }
        )
        .then(res => console.log(res.data))
    
        app.get('/axios/:id', (req, res) => {
          res.send('axios get (Restful) 传递参数' + req.params.id)
        })
    
        axios.post('http://localhost:3000/axios', {
            uname: "wxz",
            pwd: 132
        })
        .then(res => console.log(res))
    
        axios.post('http://localhost:3000/axios', 'uname=wxz&pwd=456')
        .then(res => console.log(res))

        也可以直接传对象
        var obj = new URLSearchParams();
        obj.append('uname', 'zs');
        obj.append('pwd', 123);

        axios.post('http://localhost:3000/axios', obj).then     (function (ret) {
            console.log(ret.data);

        })   
    
        app.post('/axios', (req, res) => {
          console.log(req.body);
          res.send('axios post 传递参数' + req.body.uname +     '---' + req.body.pwd)
        })   
    
    axios.interceptors.request.use(function (config) {
        return config;
    }, function (error) {
        console.log(error)
    })
    
    axios.interceptors.response.use(function (config) {
        return config;
    }, function (error) {
        console.log(error)
    }) 
    
    async function fn() {
        let p1 = await axios.get('axios', {
            params: {
                id: 'wxz'
            }
        })
        console.log(p1.data);

        let p2 = await axios.get('axios', {
            params: {
                id: 'wxz123'
            }
        })
        console.log(p2);

        let p3 = await axios.get('axios', {
            params: {
                id: 'wxz456'
            }
        })
        console.log(p3);
    }
    fn();
